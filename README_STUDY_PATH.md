# LLM 学习路线（与目录对应）

1. **01_foundations**：简介 → 词元与嵌入 → Transformer 与内部机制  
2. **02_applications**：文本分类/聚类 → 提示工程 → LangChain 与 Agent → RAG → 多模态  
3. **03_finetuning**：嵌入训练 → 分类微调 → 生成模型微调（LoRA、DPO 等）  
4. **04_code_lab**：与上面对照的可运行脚本/实验  
5. **05_resources**：书籍截图、论文、外链归档  

按章节推进时，优先在对应 `.md` 里记要点，在 `04_code_lab` 里落最小可运行示例。
